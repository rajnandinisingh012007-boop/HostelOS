# HostelOS — Lightweight Hostel Management System

HostelOS is a modern, dark-themed hostel management dashboard built with **Flask** and a **JSON-based database**. It supports three distinct roles: Admin, Warden, and Student, providing real-time tracking of residents, complaints, and visitors.

## 🚀 Quick Start

### 1. Prerequisites
Ensure you have **Python 3.x** installed on your system.

### 2. Install Dependencies
Open your terminal/command prompt in the project folder and run:
```bash
pip install flask flask-cors
